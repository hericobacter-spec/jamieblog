Jamie-blog — Next.js + MDX + Tailwind starter

Quick start

1) Install dependencies
   npm install

2) Run dev server
   npm run dev

3) Build & export (for GitHub Pages)
   npm run build
   npm run export
   # out/ directory will contain static files to publish (via gh-pages)

GitHub Actions: .github/workflows/deploy.yml will be included to deploy out/ to gh-pages branch.
